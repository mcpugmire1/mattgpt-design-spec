üìä Data Pipeline

  • Excel master sheet with 115+ STAR stories (Situation, Task, Action, Result)
  • Python script converts to JSONL format with metadata (client, role, domain, tags)
  • Automated tag generation using semantic clustering and keyword extraction
  • Data validation ensures consistency before ingestion

üß† Embeddings & Vector Search

  • Model: all-MiniLM-L6-v2 (384 dimensions, optimized for semantic similarity)
  • Chunking Strategy: Full STAR story per vector to preserve context
  • Metadata Filtering: Client, role, domain, competencies, time period
  • Top-K Retrieval: Returns 3-5 most relevant stories per query

üîç Hybrid Retrieval Strategy

  • 80% Semantic Search: Understands meaning, not just keywords (e.g., "resistance to change" matches "stakeholder buy-in")
  • 20% Keyword Matching: Ensures specific terms like "Kafka" or "SAFe" are found
  • Re-ranking Algorithm: Scores results by relevance + recency + impact
  • Fallback Logic: If no matches above threshold, suggests similar queries

üí¨ RAG with GPT-4

  • System Prompt: Defines MattGPT voice (trusted advisor, grounded in evidence)
  • Context Injection: Top 3-5 retrieved stories passed to GPT-4 as context
  • Response Structure: Direct answer ‚Üí Specific example ‚Üí Pattern/methodology ‚Üí Invitation to go deeper
  • Source Citations: Every response includes links to full STAR stories

  • Real-time semantic search with <300ms response time
  • Responsive design with mobile-first approach
